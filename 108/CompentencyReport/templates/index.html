/api/catalog

This method return the list of products as a json array.

/api/catalog/<category>

This method find the products that belong to category and return the products as a json.

/api/catalog/id/<id>

This method return the product with the specific id.

/api/catalog/cheapest

This method get the cheapest product and return it as a json.

/api/categories

This method give a list of unique categories as a json array of strings.

/api/test

This method add and remove products, get the length of the array of products, print the list and some elements.
